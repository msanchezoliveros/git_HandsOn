# seqClass.py script

## Description
The `seqClass.py` script classifies a sequence as DNA or RNA. It can also search for a specific motif within the sequence.

## Usage
```bash
python seqClass.py -s <sequence> [-m <motif>]
```

## Arguments
-s, --seq (required): Input sequence
-m, --motif (optional): Motif to search within the sequence
